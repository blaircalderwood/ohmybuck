# ohmybuck

Gatsby site for my blog/notes at [ohmybuck.com](//ohmybuck.com)

## Uses

- Gatsby
- Github Pages
- Circleci
- Cloudflare

## Developing

```sh
# Setup the correct node engine with nvm
nvm use

# Install dependencies
npm i

# Start dev server
npm start

# Deploy
git push
# Circleci should handle the rest
```
